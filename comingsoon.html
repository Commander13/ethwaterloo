My Wallet Subscriptions Transactions Coming Soon




Here are a few features that we are currently working on.


Save-O-Matic

A saving feature that would allow someone to automatically transfer certain amount of ether periodically into a deposit box that is locked until it reaches a certain amount or a certain date.


Digital Gifts

It will essentially the same as sending someone ether except that it will be embedded in a smart contract that only allows the recipient to spend the ether with a specific ethereum address.


Give to Charity

A feature that would give a user defined percentage of each deposit made into the account to a user defined charity. There are a few charities that now have ethereum addresses, notably Red Cross.
